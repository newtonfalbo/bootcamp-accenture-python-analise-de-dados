# Detecção de Anomalias em Transações

Projeto desenvolvido em Python utilizando Google Colab para identificar transações anômalas por meio de técnicas de Machine Learning.

## Tecnologias

- Python 3
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## Objetivo

Detectar possíveis fraudes ou transações incomuns utilizando algoritmos de detecção de anomalias.

## Como executar

1. Clone o repositório.
2. Abra o notebook no Google Colab.
3. Execute todas as células.
